# User Manual

Maker Splat is organized around a simple workflow:

```text
Create Project → Add Capture → Check Capture → Create 3D Scene → Preview → Export
```

## Home

The Home screen shows first-run checks and main actions.

Use it to:

- create a splat
- try the sample project
- open the guide

## Gallery

The Gallery shows your projects.

Features:

- search
- favorites
- ready projects
- deleted projects
- import ZIP

## Project Workspace

The Project Workspace is where you work on one splat.

Main actions:

- Add Capture
- Check Capture
- Clean Up Photos
- Create 3D Scene
- Preview
- Export
- Verify Output

## Capture Coach

Capture Coach reviews:

- sharpness
- lighting
- coverage
- overlap
- variety

It explains likely problems in plain language.

## Viewer

The viewer supports:

- orbit
- zoom
- pan
- point size
- background switch
- screenshot
- fullscreen

## Share Center

The Share Center creates local share links and share packages.

These are local unless you publish them elsewhere.

## Settings

Settings include:

- appearance
- performance preference
- viewer defaults
- reduced motion
- high contrast
