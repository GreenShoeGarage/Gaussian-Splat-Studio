# FAQ

## Does Maker Splat produce real `.splat` files?

It is designed to, through Real Mode, but success depends on a working Nerfstudio/COLMAP/CUDA environment.

## Does Demo Engine produce real splats?

No. Demo Engine produces previewable PLY output for normal computers.

## Is my data uploaded?

No. Maker Splat is local-first.

## Can I use video?

Yes, but for first Real Mode validation use photos.

## What is the best first object?

A matte, textured object such as a shoe, mug, toy, or small statue.

## Do I need an NVIDIA GPU?

For Real Mode, it is strongly recommended.

## What should I inspect after failure?

Start with:

- `FAILURE_CLASSIFICATION.md`
- `VALIDATION_REPORT.md`
- `REAL_MODE_REPORT.md`
- `log.txt`
