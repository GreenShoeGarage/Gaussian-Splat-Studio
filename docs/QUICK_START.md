# Quick Start

This guide gets you from zero to your first Maker Splat project.

## 1. Start the app

```bash
docker compose up --build
```

Open:

```text
http://localhost:5173
```

## 2. Create a project

Click **Create a Splat**.

Choose a name, such as:

```text
Coffee Mug
```

Use the **Recommended** preset.

## 3. Capture

Use a phone or camera.

Good first subjects:

- mug
- shoe
- toy
- plant
- matte sculpture

Avoid:

- glass
- mirrors
- shiny metal
- transparent plastic

## 4. Upload

Drag photos or video into the project.

## 5. Check Capture

Click **Check Capture**.

Read Capture Coach suggestions.

## 6. Generate

Click **Create 3D Scene**.

## 7. Preview and Export

Use the built-in viewer and download:

- `.ply`
- `.splat` when Real Mode succeeds
- MVP bundle
- diagnostics
