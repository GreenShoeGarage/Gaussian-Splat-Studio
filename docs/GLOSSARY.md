# Glossary

## Gaussian Splat

A soft 3D primitive used to represent a scene.

## SPLAT

A file format used by some Gaussian splat viewers.

## PLY

A point cloud / geometry format.

## COLMAP

Software used for camera tracking and reconstruction.

## Nerfstudio

A framework used for neural rendering and Gaussian splat training.

## Real Mode

Maker Splat mode that runs external reconstruction tools.

## Demo Engine

Fallback engine that creates previewable PLY output.

## Validation

Checks that output files appear usable.
